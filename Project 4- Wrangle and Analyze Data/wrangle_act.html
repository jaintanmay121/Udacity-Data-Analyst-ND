Quality Issues¶

twitterdf¶

  • The dataset contains retweets.
  • The tweet_id column is integer type.
  • Dataset consists of unwanted columns.
  • Source column is in HTML format with action tags.
  • Names contains values like "a", "quite".
  • Columns "name", "floofer", "doggo", "pupper" have "None" instead of NaN.
  • Standardizing the columns rating_numerator and rating_denominator into a single column rating = numerator / denominator.
  • Decimal ratings not extracted properly.

img¶

  • There are 2075 rows in the dataset which is not equal to number of tweet_ids(2356) meaning incomplete data or tweets without images.
  • Drop the irrelevant columns.
  • Various columns for dog type which could be merged into one.

tweets¶

  • Dataset consists of unwanted columns.
  • There are 2345 rows in the dataset which is not equal to number of tweet_ids(2356) meaning incomplete data.

Tidiness Issues¶

  • The columns "doggo", "pupper", "floofer" should be one column.
  • The three datasets should be one since they are related.
